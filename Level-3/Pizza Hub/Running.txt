o run the PizzaHub project manually on your local machine, follow these steps:

1. Prerequisites
Ensure you have the following installed:

Node.js (v18 or higher)
MongoDB (running locally on port 27017)
2. Backend Setup (Server)
Open a terminal and navigate to the server directory:

Install Dependencies:
bash
cd server
npm install
Seed the Database: (This populates the initial inventory and pizza varieties)
bash
node seed.js
Start the Server:
bash
npm start
The backend will be running at http://localhost:5000.
3. Frontend Setup (Client)
Open a new terminal window and navigate to the client directory:

Install Dependencies:
bash
cd client
npm install
Start the Frontend:
bash
npm run dev
The frontend will be running at http://localhost:5173 (or the port shown in your terminal).
Important Configuration (.env)
The backend relies on the .env file located in the server folder. Ensure it contains your credentials:

MONGO_URI: Your MongoDB connection string.
EMAIL_USER & EMAIL_PASS: For low-stock alerts and password resets (use a Google App Password if using Gmail).
RAZORPAY_KEY_ID & SECRET: For the payment gateway.
Summary of Commands:
Action	Directory	Command
Install All	Root	cd server; npm install; cd ../client; npm install
Seed DB	server/	node seed.js
Run Backend	server/	npm start
Run Frontend	client/	npm run dev
Once both are running, open your browser to the URL provided by the frontend terminal (usually http://localhost:5173) to start using the app!
